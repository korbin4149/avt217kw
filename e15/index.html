How to make this shape?

  1. Open Adobe Illustrator
  2. Make a new project that is 5x5 inches
  3. First, make a square, with a width and height of 1 inch
  4. Color that square #6640CC Hex code or 102,64,204 RGB
  5. drag that square up to the top left corner right on the edge. Do this for each corner, each square should be the same size and color
  6. Second, make a circle with a width and height of 1 inch
  7. Color that circle
  8. Drag that circle to the dead center of the page.
  9. The final product should be 4 squares at each corner and one circle in the middle
  10. Export it for screens, with a valid name.
